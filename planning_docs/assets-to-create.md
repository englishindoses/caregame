# Assets to Create

Working checklist for the audio phrases and images you need to generate. Tick items off as you go.

## Style notes (read first)

**For images:**
- All characters and items should share the **same illustration style**: soft cartoon, friendly, warm pastel colours, simple shapes, suitable for a 3-year-old.
- **Generate each character's 3 emotions in the same ChatGPT session**, so the character stays visually consistent (same outfit, same pose, only the face changes).
- All items on a **transparent background** (PNG with alpha) so they sit cleanly on the tray.
- The background image should leave the **centre upper area clear for the character** and the **lower 25% clear for the item tray**.

**For audio:**
- Use a **single warm, friendly narrator voice** for all characters.
- Slow and clear — the child is 3 years old.
- Slight pause between sentences within a phrase ("I'm hungry. (pause) I want some yogurt please.").
- MP3 format.

## Notes and flags

- **UK English throughout.** All phrases use UK spellings and word choices ("biscuit", "I'm sleepy", "lovely"). Keep this consistent if you record more later.

- **Voice consistency.** Whichever AI voice you pick (ElevenLabs, OpenAI TTS, etc.), generate **all phrases in one session with the same voice settings**. Switching mid-list will sound jarring.

- **If AI voices don't feel right**, record yourself. The phrase list above is what you'd read from.
