# Take Care of Me — Emotions and Phrase Variety Assets

Working checklist for the new audio phrases and images needed to add emotional range and phrase variety to the game. Tick the ☐ box on each row as you create the file.

## Style notes (read first)

### For images

- Match the existing character illustration style exactly (soft cartoon, friendly, warm pastel colours, simple shapes).
- Generate each character's new emotion in the same session as the existing images so the character stays visually consistent (same outfit, same pose, only the face changes).
- Reference the existing neutral/needy/happy images when prompting so the style, pose, and outfit match.
- Transparent PNG background.

### For audio

- Use the same warm, friendly narrator voice as the existing recordings.
- Slow and clear — the child is 3 years old.
- Slight pause between sentences within a phrase.
- MP3 format.

## Notes and flags

**UK English throughout.** Match the existing voice and spellings ('lovely', 'shall we', 'biscuit'). Keep this consistent.

**Voice consistency.** Use the same AI voice and voice settings as the original phrase set. Switching mid-list will sound jarring. If possible, regenerate one of the existing phrases in the new session first to confirm the voice still matches.

**The yawn in sleepy_1 is a sound, not a word.** Record an actual soft yawn rather than reading the word 'yawn'.

### Priority order if you're short on time

1. Thank-you variants (highest impact — plays on every success).
2. Sleepy character images (unlocks the new emotion in the game).
3. All-done and chosen variants.
4. Sad reactions.
5. Sleepy reactions.
